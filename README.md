### Neuroscience: The Three-Network Model

- **Default Mode Network (DMN)**: The "worry brain"â€”self-referential thinking, rumination, anxiety (hyperactive in depression/PTSD)
- **Task-Positive Network (TPN)**: The "focus brain"â€”present-moment awareness, execution, flow

**The Game's Intervention**: Bilateral tapping mechanically activates TPN while suppressing DMN through working memory taxationâ€”the same principle used in EMDR therapy for trauma.

1. **Bilateral Stimulation** (Andrade et al., 1997; van den Hout & Engelhard, 2012)
